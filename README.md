☕ Coffee Sales Dashboard – Excel Project

This project presents an interactive Coffee Sales Dashboard built using Microsoft Excel, designed to analyze sales performance, customer footfall, and product trends across different store locations. The dashboard transforms raw transactional data into meaningful insights to support business decision-making.

📌 Project Overview

The dashboard provides a comprehensive view of sales, customer behavior, product categories, and time-based trends. It helps identify peak hours, high-performing products, and store-level performance.
